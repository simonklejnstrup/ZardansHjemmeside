# Zardans Webpage

> Online businesscard for the band Zardans

## Introduction

This web page is for the Balkan-folk band Zardans and is done as an excersise in using Java and Spring Boot

## Features

- Embedded audio
- Embedded video
- About-page

## Technologies Used

- HTML
- Java
- Spring Boot
